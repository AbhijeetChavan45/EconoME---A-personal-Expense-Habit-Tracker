# EconoME---A-personal-Expense-Habit-Tracker
A personal finance and Habit Track  management web application to track income and expenses , Habits ,Tasks with reports ,charts and authentication.


## Features
- User authentication (Login / Signup)
- Add income and expenses
- Track transactions
- Financial reports and insights
- Clean dashboard UI
- Add habits and Task
- Snapchat Streaks Feture for completing habits


## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Database

## Purpose
The goal of this project is to help users monitor their spending habits and improve financial awareness, and also improve their habits , and easily track them
